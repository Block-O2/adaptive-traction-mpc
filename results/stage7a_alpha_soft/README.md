# Stage 7A Alpha-Soft CEM Closeout

This folder is the Stage 7A alpha-soft CEM closeout.

Alpha-soft CEM was tested and closed as a method line. It was better than the old one-step runtime filter, but the final stress validation showed mixed results and it is not strong enough to carry forward as the main safety method.

This is not the project final result. Future stages should continue with a progress governor or PSF/gatekeeper-lite, not more alpha-soft weight tuning by default.

## Contents

- `stage7a_final_report.md`: final Stage 7A alpha-soft validation report.
- `stage7a_final_summary.csv`: final validation summary table.
- `clean_theta_trajectories.png`: representative theta trajectory comparison.
- `alpha_p95_max_severity_by_method_condition.png`: alpha tail severity comparison.
- `omega_p95_max_severity_by_method_condition.png`: omega tail severity comparison.
- `target_success_treach_summary.png`: target success and reach-time summary.
- `cleanup_manifest.md`: cleanup and stage-realignment record.

No raw per-run Stage 7A logs are retained here.
